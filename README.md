# 勤怠管理アプリ

# Install

```sh
npm install -g @angular/cli
npm i
```

# Run

```sh
ng serve
```

# Deploy

```sh
npm run build-and-deploy
```

# Linting, formatting

```sh
ng lint --fix
```

## Editor

### vscode

@recommendedの通りにインストール
